# 📚 Library Management System

A console-based Library Management System developed in Java, designed to demonstrate the practical use of fundamental data structures in a real-world scenario.

---

## 🚀 Features

- 📖 Book and member management (add/remove/search)
- 🔄 Borrow and return operations
- ⏳ Waitlist system implemented with Queue (FIFO)
- ↩️ Undo functionality using Stack (LIFO)
- 🔍 Fast lookup with Hash Table and Tree structures
- ⭐ Popularity tracking using Heap / Priority Queue

---

## 🧠 Data Structures Used

| Data Structure | Purpose |
|--------------|--------|
| Linked List | Managing dynamic collections |
| Hash Table | Fast lookup by ID |
| Queue | Waitlist management |
| Stack | Undo operations |
| Tree (BST/AVL) | Efficient searching |
| Heap / Priority Queue | Tracking popular books |

---

## 📊 Example Scenario

- A user borrows a book
- If unavailable, they are added to a waitlist
- When returned, the next user automatically receives the book
- The system tracks the most popular books dynamically
